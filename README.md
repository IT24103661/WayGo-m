# WayGo – Smart Taxi & Room Management System

##  Project Description
WayGo is a centralized web application designed for tourists in Sri Lanka. 
It allows users to book taxi rides, manage tour packages, and access multiple services through a single platform.

---

## Features
- User Authentication (JWT)
- Taxi Booking System
- Tour Package Management
- Admin Dashboard
- Driver Management
- Fleet Management
- Reviews & Ratings

---

##  Technologies Used
- MongoDB Atlas (Database)
- Express.js (Backend Framework)
- React.js (Frontend)
- Node.js (Server)
